# ARRAYS (listas)

Es un contenedor con compartimientos numerados donde se pueden guardar múltiples datos bajo el mismo nombre, mantiene varios datos ordenados y son muy útiles para almacenar datos relacionados de cualquier tipo.

let nombreArray= [e0, e1, e2, e3, e4, e5]; 
let colores= ["red","orange","yellow";"green","blue"]

- para llamar alguno de los valores de Array:
  - nombreArray [n° elemento]
  - background(Colores[1]); // Esto pintara el fondo del lienzo de color anaranjado.

# CLASS 

Una clase (class) es un molde o plantilla abstracta que define la estructura, los datos (propiedades) y ls compartimientos (métodos)
